# AllHands
> *Empowering every hand in your company.*
## Description
Manage employee information, time off, holidays, positions, roles and company news in a simple easy-to-use HR system.

Built on a microservices architecture and powered by AWS cloud services (SNS, SQS, S3, SES, EC2, RDS, ElastiCache).
